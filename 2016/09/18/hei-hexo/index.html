二、安装 hexo 及相关插件

第一步先安装 hexo：

给博客安装 5 个插件：

  • hexo-generator-feed
  • hexo-renderer-ejs
  • hexo-renderer-stylus
  • hexo-generator-sitemap
  • hexo-renderer-marked

写在 package.json 文件中，通过 npm install 即可安装。

三、文件目录及配置

├── _config.yml       # 配置文件
├── package.json      # npm 配置文件
├── public            # 别鸟他
├── scaffolds         # 别鸟他
├── source            # 文章全部在这里
│   ├── CNAME         # 用于配置域名
│   ├── _posts        # 文章内容（文件夹）
│   └── about         # 可直接访问的页面
└── themes            # 主题，可在 _config.yml 中配置主题类型
    ├── jacman
    └── landscape
